THE COMPENENTS OF FUNSTION :-
1. FUNCTION NAME.
2. NUMBER OF ARGUMENTS.
3. RETURN TYPE.

1. SINGLE ROW FUNCTION :-
->SINGLE ROW FUNCTION ECECUTE ROW BY ROW.
->SRF WILL TAKE ONE INPUT AND PROCESS IT, EXECITES AND GENERATE THE OUTPUT.
->IF YOU PASS N... NO. OF INPUT THEN IT WILL GIVE  N.. NO OF OUTPUT.

2. MULTI ROW FUCTION :-
->MULTI ROW FUNCTION EXECUTES GROUP BY GORUP.
->IT TAKES ALL INPUT AT ONE SHORT AND THEN EXECUTES AND PROVIDES A SINGLE OUTPUT.
->IF PASS N.. NO. OF INPUT TO MRF IT RETURN ONLY ONE OUTPUT.
